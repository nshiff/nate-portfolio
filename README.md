# nate-portfolio

Live demo: https://nate-portfolio.onrender.com/

Portfolio site built with Google Antigravity. Demos largely built with AI assistance (Gemini Canvas, Claude).

```
# running locally
npm install
npm run dev
```

As of this writing, there are three browser-based demos:

1. A fluid simulation implementing the Navier-Stokes equations
2. A step-through Turing machine
3. A drag-and-drop modular synthesizer

... with more to come!
